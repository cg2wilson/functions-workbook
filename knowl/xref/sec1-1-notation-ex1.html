Example 1.1.10.

Let’s return to the data from Example 1.1.3.
🔗
If we let \(S\) represent for ’items sold’ and let \(d\) represent the days since the merchandise sales began, then our function is given by the notation \(S(d)\text{.}\) Here, \(d\) is the independent variable and \(S\) is the dependent variable.
🔗
The notation \(S(4)\) instructs us to look for the input \(d = 4\) and find the corresponding output paired with \(d = 4\text{.}\) In this case, \(S(4) = 16.9\) hundred items sold. Following Note 1.1.7, we could also write \(S(4) = 1690\) items sold. Knowing which one to use depends entirely on the question- be careful!
