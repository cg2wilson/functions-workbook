    Definition 1.4.2. Intercepts.

    An intercept is the location where a function crosses an axis.
    🔗
    An \(x-\)intercept is an input where the function crosses the \(x-\)axis, and a \(y-\)intercept is an output where the function crosses the \(y-\)axis.
    🔗
    🔗

    Note 1.4.3.

    Because \(x-\)intercepts occur when a function crosses the \(x-\)axis, this means that all outputs are zero. In particular, this means that all \(x-\)intercepts are solutions to the equation \(f(x) = 0\text{.}\)
    🔗
    Similarly, the \(y-\)intercept occurs at \(y = f(0)\) because the input is 0 along the \(y-\)axis. A function can only have one \(y-\)intercept by definition of a function (see Definition 1.1.2).
    🔗
    🔗

    Definition 1.4.4. Slope.

    The slope of a linear function is defined as the ratio of vertical change to horizontal change. This can be understood to be "rise over run" of the function.
    🔗
    The slope of a linear equation \(y = ax + b\) is the value of the constant \(a\text{.}\)
    🔗
    🔗

    Definition 1.4.5. Constant Rate of Change.

    The constant rate of change of a linear function is the value of its slope.
    🔗
    🔗

    Example 1.4.6.

    Consider the equation \(2x + 5y = 8\text{.}\) It is linear because we can isolate \(y\) and write the equation in the form \(y = ax + b\text{;}\) rewriting gives
    \begin{gather*} y = -\dfrac{2}{5}x + \dfrac{8}{5} \end{gather*}
    🔗
    If we want to find the \(x-\)intercept, we can do so by setting \(y = f(x) = 0\text{.}\) This gives:
    \begin{align*} 0 \amp = -\dfrac{2}{5}x + \dfrac{8}{5}\\ \dfrac{2}{5}x \amp = \dfrac{8}{5} \\ x \amp = 4 \end{align*}
    So the \(x-\)intercept occurs at \((4,0)\text{.}\)
    🔗
    For the \(y-\)intercept, we need to find \(f(0)\text{.}\) This means we plug in 0 to the equation:
    \begin{gather*} f(0) = -\dfrac{2}{5}(0) + \dfrac{8}{5} = \dfrac{8}{5} \end{gather*}
    🔗
    Since the slope is the value of the constant \(a\) when the equation is in the form \(y = ax + b\text{,}\) and we know that \(y = -\dfrac{2}{5}x + \dfrac{8}{5}\text{,}\) the slope is \(a = -\dfrac{2}{5}\text{.}\) This tells us that when creating the line, to move from one point to another, we must go down two units and right five units.
