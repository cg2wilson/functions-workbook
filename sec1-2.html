    Definition 1.2.2. Complete Graph.

    A complete graph is a graph which shows the basic shape of the graph and important points on the graph (including points where the graph crosses the axes and points where the graph turns) and suggests what the unseen portions of the graph will be.
    🔗
    🔗

    Note 1.2.8.

    The location where a graph hits a "peak" or a "trough" has a name! These are called relative extrema.
    🔗
    🔗

    Definition 1.2.9. Relative Maximum/Relative Minimum.

    A relative maximum of a function \(f(x)\) is a point on the graph of the function where the output is larger than the outputs around it.
    🔗
    A relative minimum of a function \(f(x)\) is a point on the graph of the function where the output is smaller than the outputs around it.
    🔗
    Collectively, relative maxima and minima are called relative (or local) extrema
    🔗
    🔗

    Note 1.2.10.

    Because the words are derived from Latin, the plural of "maximum" is "maxima", the plural of "minimum" is "minima", and the plural of "extremum" is "extrema".
    🔗
    🔗

    Definition 1.2.15. Roots/Zeros/Solutions.

    A root of a function is the input value where it crosses the \(x-\)axis, i.e. is a solution to the equation \(f(x) = 0\text{.}\)
    🔗
    A root is also called a zero or solution.
    🔗
    🔗
