    Section 1.1 Section 1.1

    Definition 1.1.1. Relation.

    A relation is a rule which establishes a relationship between two variables.
    🔗
    🔗

    Definition 1.1.2. Function.

    A function is a relation that assigns to each element of one set (called the domain) exactly one element of the a second set (called the range).
    🔗
    Elements of the domain are called inputs or independent variables and elements of the range are called outputs or dependent variables.
    🔗
    A function may be defined by a set of ordered pairs, a table, a graph, an equation, or a verbal description.
    🔗
    🔗

    Example 1.1.9.

    The following are all examples of functions:
    • \(\displaystyle y = 3x + 5\)
      🔗
      🔗
    • \(\displaystyle v = t^2 - 10\)
      🔗
      🔗
    • \(\displaystyle y = 0.2^x\)
      🔗
      🔗
    🔗
    You can check that these are functions because every input has only one output. But how do we know which is the input and which is the output?
    🔗
    The output is usually written as either a single variable (like \(y\) or \(v\) above) or by using function notation.
    🔗
    🔗

    Function Notation.

    For a function whose inputs are represented by the variable \(x\text{,}\) and whose outputs are represented by the variable \(f\text{,}\) the the function notation for this relationship is shown as
    \begin{gather*} f(x) \end{gather*}
    🔗
    This is read as "\(f\) of \(x\)".
    🔗
    🔗

    Example 1.1.10.

    Let’s return to the data from Example 1.1.3.
    🔗
    If we let \(S\) represent for ’items sold’ and let \(d\) represent the days since the merchandise sales began, then our function is given by the notation \(S(d)\text{.}\) Here, \(d\) is the independent variable and \(S\) is the dependent variable.
    🔗
    The notation \(S(4)\) instructs us to look for the input \(d = 4\) and find the corresponding output paired with \(d = 4\text{.}\) In this case, \(S(4) = 16.9\) hundred items sold. Following Note 1.1.7, we could also write \(S(4) = 1690\) items sold. Knowing which one to use depends entirely on the question- be careful!
    🔗
    🔗

    Example 1.1.13.

    We can also find function values when given a rule or graph.
    🔗
    Consider the function \(f(x) = x^4 - 4x^3 + 2\text{.}\) The statement \(f(1)\) says that the input variable, which we represent by \(x\text{,}\) should be replaced by the number \(1\text{.}\) So,
    \begin{gather*} f(1) = (1)^4 - 4(1)^3 + 2 = 1 - 4 + 2 = -1 \end{gather*}
    Thus, \(f(1) = -1\)
    🔗
    We can use the graph to find similar information.
    🔗
    We can find \(f(2)\) by first finding \(2\) on the horizontal axis, then finding the corresponding point on the function.
    🔗
    🔗
